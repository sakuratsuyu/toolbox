# Calibration Formatter

Support format
- colmap
- NeRF Synthetic / nerfstudio
- MVSNet
- easyvolcap

## 坐标系转换

### OpenGL

- x points right, y points up, z points backwards scene.

- 使用者
    - Blender
    - NeRF synthetic

### OpenCV

- x points right, y points down, z points towards scene (forward).

- 使用者
    - COLMAP
